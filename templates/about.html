About SmartCart

SmartCart is a full-stack e-commerce web application designed to deliver a smooth, secure, and efficient online shopping experience. It simplifies both product management for administrators and shopping convenience for users.

The platform features a secure authentication system with OTP-based email verification, ensuring enhanced security and protecting user data. Administrators can easily manage products, track inventory, and monitor system activity through an intuitive dashboard.

Built using Flask for backend development, MySQL for database management, and Bootstrap for responsive design, SmartCart demonstrates modern web development practices and scalable architecture.

The goal of SmartCart is to provide a reliable, secure, and user-friendly e-commerce management solution that can be expanded with advanced features such as payment gateway integration, invoice generation, cart management, and order tracking.
